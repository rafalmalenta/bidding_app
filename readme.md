Bidding app made with symfony not finished yet

###requirements
* php 7.3+
* composer
###installation
* clone or download
* run 'composer install'
* then run 'php -S 127.0.0.1:8000 -t public'*
* navigate to 'http://127.0.0.1:8000/' in your browser
